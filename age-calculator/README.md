### Age Calculator

Age  Calculator website where you can input your birth date and get the Age
in days,months and years.

### Use of the Project:
A person's age can be calculated in days,months and years. It can be used 
in offices,malls etc.

### Used Technologies
  * HTML5
  * CSS3
  * JavaScript
