# Temperature Convertor

This webpage converts temperatures from and to Celcius, Fahrenheit and Kelvin.

### Tech Stack:

- HTML5
- Bulma CSS
- Vanilla JS

### Prequisites:

- Git and GitHub installed in the machine in order to clone the repo.
