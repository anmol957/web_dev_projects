# Image Finder - React Application   

## About the Project
A single page web application that uses the PixaBay API to display the images according to user search.

## Technologies Used

- React
  - Hooks
  - Components
- CSS
- Javascript
  - Axios
  - ES6
- API
- MaterialUI


## API Used
PixaBay API

Link : https://pixabay.com/api/
